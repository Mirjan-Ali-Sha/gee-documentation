What is Google Earth Engine?
=============================

Google Earth Engine is a cloud computing platform for planetary-scale geospatial analysis that uses Google's computing infrastructure to process very large geospatial datasets.

Key Features:
=============

* Petabyte-scale data catalog
* Cloud-based processing
* JavaScript and Python APIs
* Interactive development environment
* Machine learning capabilities
